Q: Can I use multiple timers at once?
A: No - only use Study Timer OR Sleep Timer,
   not both. They serve different purposes.

Q: Does Loop Mode work with timers?
A: YES! Loop Mode keeps going until timer stops
   it. Perfect combination!

Q: Does Repeat Mode slow down my session?
A: Yes - if set to 3 times, it triples the time
   per card. Plan accordingly!

Q: Can I change settings during auto-play?
A: Some settings (sliders) update live. Others
   (checkboxes) need stop/restart. Best to set
   everything before starting!

Q: Does Session Stats track manual card reviews?
A: No - only tracks auto-play sessions. Stats
   start when you click Auto-Play.

Q: Will Focus Mode work on mobile?
A: YES! Even better on mobile - true fullscreen
   experience with no browser UI!

Q: Can I use Focus Mode without Auto-Play?
A: Yes, but it's more useful with Auto-Play so
   you don't need to touch the screen.

Q: Do settings save between sessions?
A: No - they reset when you close the page. Set
   your preferences each time (takes 30 seconds).
